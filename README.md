# Catalog Lending App for Sports Equipment

## A Heroku web application designed using the Django framework where users can upload sports equipment; other users can then request these equipment to be lent to them and give ratings after use!

## Overview:
Seeing that there are few sources available for lending sports equipment online, this Catalog Lending App was created to help serve that purpose! This was designed with an audience of college students in mind, where people may have conflicting schedules and a lower budget to justify buying sports equipment of their own to buy and use when they please.

On this web application, users can register as either a Patron or a Librarian:
- Patrons are users who plan to borrow sports equipment from others.
- Librarians are users who plan to upload sports equipment for other users to borrow. These users can also promote Patrons to be Librarians.
- Both users can create Collections and place various sports equipment in them to assist users in searching for them.
  
You can search for equipment by various categories such as Collection or by pre-defined categories like the Sport (Basketball, Tennis, etc.) or the type of Exercise (Weight Lifting, etc.).

As mentioned, you can also create Collections:
- Patrons can create Public Collections, which are accessible only by that user and by any Librarian.
- Librarians can create Public Collections or Private Collections. Private Collections are accessible by any other Librarians who are added as contributors to that collection.

## Creating an account and using the application:
- In order to use the application, you need to create an account, which requires you to sign in using a Google account.
- You can also provide a username and a profile picture for your account.
- As a Patron, you can search for a piece of equipment you would like to borrow and send a request for it.
- Then, the Librarian who owns that equipment can choose who to lend it to.
- When the Patron is done using the equipment, the Patron can provide a rating (0 to 5 stars) for the quality of the equipment.
- The average rating on a piece of equipment can help users gauge whether or not they would like to request it.

## Disclaimer
- This app should be considered as a prototype since it is not fully designed for public deployment. As such, the app should not currently be used for personal use. There are some small security issues with the application, as well.
